# HabitFlow

HabitFlow es una página web sencilla de prueba para presentar una aplicación ficticia de seguimiento de hábitos diarios. El proyecto está pensado como ejemplo básico de documentación para una web estática formada por tres archivos principales: `index.html`, `main.js` y `styles.css`.

## Descripción del proyecto

La página muestra una landing page sencilla donde el usuario puede consultar una presentación breve de la aplicación, ver una lista de hábitos recomendados y marcar algunos elementos como completados mediante JavaScript.

Este proyecto no necesita backend, base de datos ni instalación de dependencias. Está creado únicamente con HTML, CSS y JavaScript puro.

## Estructura de archivos

```text
habitflow/
├── index.html
├── main.js
└── styles.css
```

### `index.html`

Contiene la estructura principal de la página web:

- Encabezado con el nombre del proyecto.
- Sección principal con una descripción de la aplicación.
- Lista de hábitos diarios.
- Botones o elementos interactivos conectados con JavaScript.

### `main.js`

Contiene la lógica básica de interacción:

- Detectar clics en los botones.
- Marcar hábitos como completados.
- Mostrar mensajes simples al usuario.
- Modificar clases CSS de forma dinámica.

### `styles.css`

Contiene los estilos visuales de la página:

- Colores principales.
- Diseño general de la landing page.
- Estilos de botones, tarjetas y listas.
- Adaptación básica para pantallas pequeñas.

## Cómo ejecutar el proyecto

No es necesario instalar nada. Solo tienes que abrir el archivo `index.html` en tu navegador.

También puedes usar una extensión como Live Server en Visual Studio Code para ver los cambios automáticamente mientras editas el proyecto.

## Pasos recomendados

1. Descarga o clona el proyecto.
2. Abre la carpeta en Visual Studio Code.
3. Abre `index.html` directamente en el navegador o ejecútalo con Live Server.
4. Modifica `styles.css` para cambiar el diseño.
5. Modifica `main.js` para añadir más interactividad.

## Ejemplo de uso

Al abrir la página, el usuario verá una lista de hábitos como:

- Beber agua.
- Leer 10 minutos.
- Salir a caminar.
- Organizar la agenda del día.

Cuando el usuario haga clic sobre un hábito, este podrá marcarse visualmente como completado.
